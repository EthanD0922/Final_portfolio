Summary:
Friend Finder is an application that asks the user a series of questions and then matches them with someone that has simular likes an interests to connect them as friends.
Tech Used:
HTML, Bootstrap, Javascript, Jquery, Path, Express, Heroku.
Function:
I used express to create a basic server that is hosted through Heroku. This app uses Javascript to tally up the results of the Survey and then pass them through my API. Using a double for-loop I comb through each entry in the database and compare the user with each one until the best connection is found. I then push the users information back to the front where it is shown as a modal. The user is then saved to the database for future use.
